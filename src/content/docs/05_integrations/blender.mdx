The Shale add-on for Blender integrates the review pipeline directly into your modeling and rendering workflow.

## Requirements

- Blender 4.xxx (Does not work with 5.xx yet)
- Shale account with an active organization

## Installation

1. Copy this link https://plugins.shalereview.com/blender/index.json
2. In Blender, go to Edit → Preferences → Extensions → Install
    - We've noticed that the extensions window slows down most computers, this is unrelated to Shale and is an issue with Blender. 
3. Add a remote repository using the + button in the top right
4. Paste the link above into the remote repository unrelated
5. The Shale Plugin will now show up in the `N` view 

For more detailed instructions refer to the [Upload Via Plugin](/03_plugin_upload/overview/)
